#  1. 프로젝트 소개
## 프로젝트 목표
    1. Arduino와 pyqt를 모두 사용할 수 있는 스마트 우편함 시스템을 설계
    2. MQTT를 통한 WIFI 통신
    3. 센서들을 통한 실시간 데이터로 데이터베이스 구축
    
## 1.2 프로젝트 팀 소개
    김영환 - 리더 / Python / PYQT / HW 
    김진홍 - PYQT / Database 
    과악준 - Arduino / MQTT
    유겸희 - Arduino / HW
